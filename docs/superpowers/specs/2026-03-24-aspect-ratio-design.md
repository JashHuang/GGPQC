# 圖片尺寸設定功能設計規格

**最後更新日期：** 2026-03-24

## 1. 功能概述

在 V6 首頁新增圖片尺寸快捷選擇器，讓用戶可以選擇生成早安圖的長寬比。

### 成功標準

- 用戶可以在首頁快速切換 3 種長寬比
- 選擇的尺寸會被記憶，下次自動套用
- 不影響原本的一鍵生成流程

---

## 2. 支援的長寬比

| 比例 | 尺寸 (px) | 用途 |
|------|-----------|------|
| 1:1 (方形) | 1080×1080 | LINE 分享，長輩最習慣 |
| 9:16 (直式) | 1080×1920 | 手機牆紙、限時動態 |
| 4:3 (標準) | 1080×810 | 接近電腦螢幕比例 |

## 4. 實作細節

### 4.1 資料結構

```typescript
type AspectRatio = '1:1' | '9:16' | '4:3';

type Settings = {
  // ... existing settings
  aspectRatio: AspectRatio;
};
```

### 4.2 尺寸對應表

```javascript
const ASPECT_RATIOS = {
  '1:1': { width: 1080, height: 1080 },
  '9:16': { width: 1080, height: 1920 },
  '4:3': { width: 1080, height: 810 },
};
```

### 4.3 Canvas 生成邏輯

修改 `useAutoGenerate.js` 中的 `createComposedImage` 函式：
- 根據 `settings.aspectRatio` 動態設定 canvas 尺寸
- 調整文字排版邏輯以適應不同比例
